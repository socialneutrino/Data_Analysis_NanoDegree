We can go further in our analysis of what this data means and calculate if there is a rank correlation between the fare price, and the passenger class - treating class as a ordinal variable, since passenger class is clearly not a linear scale. It is questionable about whether we can do this, since you could call class a categoric variable.

For the sake of this assignment, I am going to assume that passenger class is an ordinal variable, a sign of increasing socioeconoic status, that increases as the class number reduces. I can analyse the association with passenger fare by using Spearman's Rank Correlation.

Performing Spearman's Rank Correlation hypothesis test for association between fare and passenger class¶

Since class is an integer, we could treat it as an ordinal variable of socioeconomic heirarchy. How strong is the link between class and fare price.

Null and alternative hypotheses¶

  • $H_0$: The null hypothesis is that there is no association between fare price and passenger class
  • $H_1$: The alternative hypothesis is that there is an association between fare price and passenger class.

Setting $\alpha = 0.05$, we have a confidence level of 95% for our two-tailed hypothesis test The critical p-values of the test returned will be less than 0.05

The Spearman test shows that is a very low p value (see below) so we can reject $H_0$, and accept $H_1$ - there is an association between fare price and passenger class.

However, the r value is not that large. As we can see from the histogram, there is a fair amount of overlap of fare prices between the passenger class groups.

A look at fare and correlation with survival¶

To analyse this data, I have standardized the Fare variable. I have then done a crude measure of correlation of survival with fare price. Since survival is 1 or 0, I can measure a correlation by looking at the linear regression and calculating Pearson's MCC. This is not strictly statistically acceptable, since survival is not a continuous variable, but it is much easier than simple logistic regression, which I believe is the more statistically acceptable way to find an assoication between a binary dependent variable and a continuous independent variable.
